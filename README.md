🎵 Discord Music Bot

A powerful and simple Discord music bot built to play high quality music directly in your voice channels.

✨ Features

- ▶️ Play music from YouTube using a link or search
- ⏭ Skip songs
- ⏹ Stop music
- 🔁 Loop songs
- 📃 Queue system
- 🔊 High quality audio
- ⚡ Fast and lightweight
- 🎧 Easy to use commands

📜 Commands

Command| Description
"play <song name / link>"| Play a song from YouTube
"skip"| Skip the current song
"stop"| Stop the music and clear the queue
"queue"| Show the current music queue
"loop"| Repeat the current song
"pause"| Pause the music
"resume"| Resume the music

🚀 Installation

1. Clone the repository
2. Install dependencies
3. Add your bot token
4. Run the bot

Example:

git clone https://discord.gg/aven
cd music-bot
npm install
node index.js

⚙️ Requirements

- Node.js 16 or higher
- Discord Bot Token
- FFmpeg installed

🔧 Configuration

Open the configuration file and add your Discord bot token:

TOKEN=YOUR_BOT_TOKEN

📌 Notes

Make sure the bot has the following permissions:

- Connect to Voice Channel
- Speak in Voice Channel
- Send Messages
- Read Messages
